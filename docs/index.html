Frequently asked questions

Quick answers before you download.

How does the free trial work?
Download the installer or portable build and run it — you get 14 days with full features, no account or credit card. When the trial ends, smooth scrolling pauses until you enter a license key.
What's the difference between installer and portable?
The installer adds Smoothly to the Start Menu and is the recommended option for most users. The portable build is a single .exe you can run from any folder without installing — handy for USB drives or locked-down setups.
Windows SmartScreen blocked the download — is it safe?
New or niche apps often trigger SmartScreen until they build reputation. If you trust this download from our official page, click More info → Run anyway. Always download only from this site or your Lemon Squeezy purchase link.
How do I activate my license?
After purchase you'll receive a key by email (format SMOOTH.xxx.yyy). Open Smoothly → License → paste the key → Activate license. The license is tied to your PC and works offline.
Which apps does Smoothly work with?
Smoothly hooks mouse-wheel input system-wide. It works in Chrome, Edge, Firefox, VS Code, Cursor, Office, PDF readers, and most desktop apps. Some games or apps that capture the mouse exclusively may not be affected.
Can I turn smooth scrolling off quickly?
Yes — use the toggle in the settings window or the default global hotkey Ctrl + Shift + S to switch back to native scrolling instantly.
Where is my settings file stored?
Configuration is saved locally at %AppData%\Smoothly\settings.json. Uninstalling does not remove this file unless you delete it manually.
