Smoothly hero demo video — add these files when ready:

  docs/assets/demo.webm   (primary, VP9, target < 3 MB)
  docs/assets/demo.mp4    (fallback H.264, target < 5 MB)

Recording (15–25 s loop):
  1. Chrome/VS Code with native choppy scroll (~2 s)
  2. Enable Smoothly from tray (~1 s)
  3. Same scroll, now smooth (~5–8 s)
  4. Settings panel, smoothness slider (~3 s)
  5. Back to smooth scroll to close the loop (~3 s)

Tools: ShareX or Win+G (Xbox Game Bar). 1280×720 or 1920×1080, 30 fps, cursor visible, no audio.

Compress (ffmpeg):
  ffmpeg -i demo-raw.mp4 -c:v libvpx-vp9 -crf 35 -b:v 0 demo.webm
  ffmpeg -i demo-raw.mp4 -c:v libx264 -crf 28 -movflags +faststart demo.mp4
  ffmpeg -i demo.mp4 -ss 00:00:01 -vframes 1 demo-poster.png

Until then, demo-poster.png (copy of og-image) is shown in the hero card.
