# Smoothly — public website

Marketing landing and trial downloads for [Smoothly](https://smoothly.lemonsqueezy.com/buy). **No application source code** lives in this repository.

## GitHub Pages

1. Create a public repo named **`Smoothly-site`** on GitHub (or push this folder to it).
2. **Settings → Pages → Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **`main`** / folder **`/docs`**
3. Site URL: **https://cristianjavierdacamarasousa.github.io/Smoothly-site/**

After each app release, update [`docs/version.json`](docs/version.json) with Lemon Squeezy download URLs (see [docs/LEMON-SQUEEZY.md](docs/LEMON-SQUEEZY.md)).
